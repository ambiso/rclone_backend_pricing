# rclone_backend_pricing
Pricing calculator for backends supported by rclone

# Dependencies

## Node

Install [nvm](https://github.com/nvm-sh/nvm)
and run

```
nvm install node
```

put script it outputs into your .bashrc (or equivalent)

## Parcel Bundler

```
npm i -g parcel-bundler
```

# Build

```
make watch
```

# Deploy

```
make deploy
```
